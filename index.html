Browser-Erweiterung

Notizen direkt für Webseiten speichern.

TabNotes hilft dir dabei, Notizen, Ideen, markierte Texte und wichtige Informationen direkt mit Webseiten zu verknüpfen. Ideal für Recherche, Lernen, Online-Shopping, YouTube-Videos und tägliches Browsen.

Was TabNotes kann

🌐 Webseiten-Notizen Speichere Notizen passend zur aktuell geöffneten Webseite.
📝 Mehrere Notizen pro Seite Erstelle verschiedene Notizen für dieselbe Webseite.
🖱️ Markierten Text speichern Übernimm markierten Text per Rechtsklick in TabNotes.
📌 Favoriten & Pins Behalte wichtige Notizen schnell im Blick.
🗑️ Papierkorb Stelle gelöschte Notizen wieder her.
🌙 Dark Mode Arbeite angenehmer mit dunkler Oberfläche.
📤 Export & Backup Exportiere Notizen als Markdown oder sichere sie als Backup.
▶️ YouTube-Unterstützung Erstelle Notizen zu Videos und Lerninhalten.

Typische Nutzung

Öffne eine Webseite, ein Video, einen Artikel oder ein Produkt.
Schreibe direkt eine Notiz oder speichere markierten Text per Rechtsklick.
Organisiere wichtige Notizen mit Favoriten, Pins und Papierkorb.

Support

Hilfe & häufige Fragen

Wo werden meine Notizen gespeichert?

Deine Notizen werden lokal im Browser gespeichert. TabNotes überträgt keine Notizen an externe Server.

Kann ich meine Notizen sichern?

Ja. Über das Mehr-Menü kannst du ein Backup erstellen und später wieder importieren.

Kann ich markierten Text speichern?

Ja. Markiere Text auf einer Webseite, öffne das Rechtsklick-Menü und wähle „Zu TabNotes hinzufügen“.

Warum funktioniert TabNotes nicht auf chrome:// Seiten?

Browser-interne Seiten wie chrome:// dürfen aus Sicherheitsgründen nicht von Erweiterungen verändert werden.

Wie kann ich Support kontaktieren?

Support-Anfragen können über den Chrome Web Store Eintrag von TabNotes gestellt werden.

Datenschutz

Datenschutzerklärung

TabNotes speichert alle Notizen ausschließlich lokal im Browser des Nutzers. Es werden keine Daten an externe Server übertragen.

Lokale Speicherung

TabNotes speichert Notizen, Einstellungen, Favoriten, angepinnte Notizen und Papierkorb-Einträge lokal im Browser des Nutzers.

Keine Datenerfassung

TabNotes erhebt, überträgt, verkauft oder teilt keine personenbezogenen Daten.

Kein Tracking

TabNotes verwendet keine Analyse-Dienste, kein Tracking und keine externen Server.

Browser-Berechtigungen

TabNotes nutzt Browser-Berechtigungen ausschließlich, um die Seitenleiste auf Webseiten anzuzeigen, Notizen lokal zu speichern und markierte Texte per Kontextmenü zu übernehmen.
