TabNotes

Notizen für jede Webseite.

TabNotes ist eine Browser-Erweiterung zum Speichern von Notizen direkt auf Webseiten. Ideal für Recherche, Lernen, Shopping, Videos und tägliches Browsen.

Funktionen

🌐 Webseiten-Notizen Speichere Notizen direkt passend zur aktuell geöffneten Webseite.
📝 Mehrere Notizen pro Seite Erstelle verschiedene Notizen für dieselbe Webseite.
🖱️ Markierten Text speichern Übernimm markierten Text direkt per Rechtsklick in TabNotes.
📌 Favoriten & angepinnte Notizen Behalte wichtige Notizen schnell im Blick.
🗑️ Papierkorb Gelöschte Notizen können wiederhergestellt werden.
🌙 Dark Mode Eine dunkle Oberfläche für angenehmes Arbeiten.
📤 Export & Backup Exportiere Notizen als Markdown oder sichere sie als Backup.
▶️ YouTube-Unterstützung Erstelle Notizen zu Videos und Lerninhalten.

Datenschutzerklärung

TabNotes speichert Notizen, Einstellungen, Favoriten, angepinnte Notizen und Papierkorb-Einträge ausschließlich lokal im Browser des Nutzers.

  • Kein Konto erforderlich
  • Keine Analyse-Dienste
  • Kein Tracking
  • Keine externen Server
  • Keine Weitergabe von Daten an Dritte

Browser-Berechtigungen

TabNotes nutzt Browser-Berechtigungen ausschließlich, um die Seitenleiste auf Webseiten anzuzeigen, Notizen lokal zu speichern und markierte Texte per Kontextmenü zu übernehmen.
